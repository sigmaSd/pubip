IP Address API

A simple API to retrieve your public IP address.

Endpoint

GET /api

Response

{
  "ip": "203.0.113.42"
}

Example Usage

cURL

curl {{ORIGIN}}/api

JavaScript (fetch)

fetch('{{ORIGIN}}/api')
  .then(response => response.json())
  .then(data => console.log('Your IP:', data.ip));

Python

import requests
response = requests.get('{{ORIGIN}}/api')
print(response.json()['ip'])
Privacy Note: Your actual IP address is not stored or logged by this service. We hash IP addresses with a cryptographic salt for anonymous statistics only. The hashed values cannot be reversed to reveal your original IP address.

Note: While this service doesn't log your IP, Cloudflare (the infrastructure provider) may have access to connection metadata as part of their CDN services.

CORS

This API supports Cross-Origin Resource Sharing (CORS), so you can call it from any domain.
